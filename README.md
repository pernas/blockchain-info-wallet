# blockchain-info-wallet

## Requirements

- `brew install stack` (Mac OS)

## Development

- clone
- cd project dir
- stack build
- stack exec blockchain your-guid

## Installation

- clone
- cd project dir
- stack install
- add home/.local/bin to your path
- now you can run `> blockchain your-guid` from anywhere
